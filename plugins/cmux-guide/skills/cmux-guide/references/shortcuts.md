# cmux キーボードショートカット

すべてのショートカットは `~/.config/cmux/cmux.json` の `shortcuts.bindings` でカスタマイズ可能。
2段階コードは配列で指定: `["ctrl+b", "c"]`。
無効化: 空文字列, `null`, `"none"`, `"clear"`, `"unbound"`, `"disabled"`。

## アプリ

- ⌘, — 設定
- ⌘⇧, — 構成再読み込み
- ⌃⌥⌘. — 全cmuxウィンドウ表示/非表示
- ⌥⌘F — グローバル検索
- ⌘⇧P — コマンドパレット
- ⌃N / ⌃P — パレット次/前の結果
- ⌘⇧N — 新規ウィンドウ
- ⌃⌘W — ウィンドウを閉じる
- ⌃⌘F — フルスクリーン
- ⌘⇧O — 前回セッション再開
- ⌘Q — 終了

## ワークスペース

- ⌘B — 左サイドバー切り替え（ワークスペース一覧）
- ⌥⌘B — 右サイドバー切り替え（File Explorer: Finderライクなディレクトリツリー）
- ⌘N — 新規ワークスペース
- ⌘O — フォルダを開く
- ⌘P — ワークスペーススイッチャー
- ⌃⌘] / ⌃⌘[ — 次/前のワークスペース
- ⌘1–9 — ワークスペース選択
- ⌘⇧R — ワークスペース名変更
- ⌥⌘E — 説明編集
- ⌘⇧E — 右サイドバー（File Explorer）フォーカス
- J/K/⌃N/P — サイドバー行移動。File ExplorerではH/Lでフォルダ展開/折りたたみ、/で検索
- ⌘⇧W — ワークスペースを閉じる

## サーフェス（タブ）

- ⌘T — 新規サーフェス
- ⌘⇧] / ⌘⇧[ — 次/前のサーフェス
- ⌃1–9 — サーフェス選択
- ⌘R — タブ名変更
- ⌘W — タブを閉じる
- ⌥⌘T — 他のタブを閉じる
- ⌘⇧T — 閉じたブラウザパネルを復元
- ⌘⇧M — コピーモード切り替え
- ⌘S — ファイルプレビュー保存

## 分割ペイン

- ⌥⌘←→↑↓ — ペインフォーカス移動
- ⌘D — 右に分割
- ⌘⇧D — 下に分割
- ⌥⌘D — 右にブラウザ分割
- ⌥⌘⇧D — 下にブラウザ分割
- ⌘⇧↩ — ペインズーム切り替え
- ⌃⌘= — 分割サイズ均等化

## ブラウザ

- ⌘⇧L — ブラウザを開く
- ⌘L — アドレスバーフォーカス
- ⌘[ / ⌘] — 戻る/進む
- ⌘R — ページ再読み込み
- ⌘= / ⌘- / ⌘0 — 拡大/縮小/実寸
- ⌥⌘I — 開発者ツール
- ⌥⌘C — JavaScriptコンソール
- ⌘⇧G — React Grab

## 検索

- ⌘F — 検索
- ⌘⇧F — ディレクトリ内検索
- ⌘G / ⌥⌘G — 次/前を検索
- ⌥⌘⇧F — 検索バー非表示
- ⌘E — 選択範囲で検索

## 通知

- ⌘I — 通知パネル
- ⌘⇧U — 最新未読へ移動
- ⌥⌘U — 未読状態切り替え
- ⌃⌘U — 最古未読→次の最新未読
- ⌘⇧H — フォーカス中パネルフラッシュ
